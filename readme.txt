Visit  : www.mk2k.net

This is a port of the Chipmunk Physics including the demonstrations that come
with it to the PSP. For graphics and user input the JGE++ Engine by Dr. Watson
has been used.

- -- --------------------------------------------------------------------- -- -
I    Installation
- -- --------------------------------------------------------------------- -- -
I.1 Binaries Installation
Original Firmware 1.50 users should copy the content of Binary/PSP/FW150 to the
PSP/GAME directory on the memorystick.
Custom Firmare users should copy the content of Binary/PSP/CFW3xx to the
PSP/GAME directory on the memorystick.
Windows users can take a look into Binary/Win32.

I.2 Source Installation
To have a quick start on the source, get JGE++ first:
- http://jge.googlecode.com/files/JGE_1_0.part1.rar
- http://jge.googlecode.com/files/JGE_1_0.part2.rar
extract it and put the Chipmunk_Demo directory from the Source directory to
JGE/Projects

- -- --------------------------------------------------------------------- -- -
II   PSP Controls
- -- --------------------------------------------------------------------- -- -

Left/Right Shoulder Buttons: Switch from Demo 1 up to Demo 7 and vice versa
D-Pad Down                 : Switch from animation to stop-motion
D-Pad Right                : - Animate in stop-motion mode when pressed
                             - Frameskip = 1 in animation mode
D-Pad Left                 : Advance one frame
Triangle Button            : Start/Stop taking screen capture (slows the whole
                             animation down)
Start                      : Restart the current Demo
Select                     : Switch from 4:3 to 16:9 aspect ratio
